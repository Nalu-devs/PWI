Formularios e botões

Tags básicas do formulario: Form, Action e method

Tag Form: É o elemento fundamental que envolve todos os outros elementos relacionados ao formulário. Ela define a área onde os usuários podem inserir dados e como esses dados serão enviados para o servidor.

Tag Action: Especifica a URL (endereço) para onde os dados do formulário serão enviados quando o usuário o submeter.

Tag Method: Define o método HTTP usado para enviar os dados do formulário para o servidor.

Os dois mais usados: Get e POST

Tag Name: Define um nome para tabela, útil caso queira usar os dados dentro de PHP ou Javascrip

Tag input: Abre uma aba para o usuario poder digitar, onde pode ter diversos tipos de dados

Tag type: Serve para declarar o tipo de dado que será digitado nele

Type text: Declara tipo texto

Type password: Declara tipo senha

Type radio: Declara tipo escolha única
