Tabela

Tags principais: table, tr, td, th, border, thead e tbody

Tag Table: Define que o que for escrito dentro dela será em uma tabela de colunas e linhas

Tag Tr: Define uma linha da tabela

Tag Td: Define uma celula (coluna) da tabela

Tag Th: Define uma celula em negrito

Tag Border: Define uma borda para a tabela

Tag Thead: Define o que vai no cabeçalho da tabela

Tag Tbody: Define o que vai no corpo da tabela

Exemplo de tabela
